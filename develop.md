## Tailwind

* [nvm](https://github.com/nvm-sh/nvm)
* [Tailwind CSS](https://tailwindcss.com/)
* [daisyui](https://daisyui.com/)

Configure tailwind vscode extension:

```json
"tailwindCSS.includeLanguages": {
    "templ": "gohtmltmpl"
},
"files.associations": {
    "*.css": "tailwindcss"
},
"editor.quickSuggestions": {
    "strings": "on"
}
```
